# 隐私政策 / Privacy Policy

爱奇艺实时弹幕转发器 (iQiyi Danmu Forwarder)

最后更新：2026-07-16

本扩展尊重并保护所有用户的隐私。

## 我们收集哪些数据

不收集任何数据。本扩展不会收集、存储或传输任何个人身份信息、
浏览记录、网页内容或用户通信。

## 本地存储

本扩展仅使用 `chrome.storage.local` 在你的浏览器本地保存扩展设置
（如转发间隔、间隔模式、过滤选项、面板位置、开关状态）。这些数据
永远不会离开你的设备，也不会发送到任何服务器。

## 权限说明

- **storage**：用于在本地保存上述扩展设置。
- **对 `*.iqiyi.com` 的访问**：仅用于在爱奇艺视频页面读取弹幕文本
  并发送弹幕，以实现扩展的核心功能。访问范围严格限制在爱奇艺域名，
  不涉及任何其他网站。

## 第三方

本扩展不包含任何第三方分析、广告或追踪代码，也不加载任何远程代码。

## 联系方式

如有疑问，请通过项目 GitHub 仓库提交 issue：
https://github.com/wenzier/iqiyi-danmu-forwarder

# Privacy Policy (English)

iQiyi Danmu Forwarder

Last updated: 2026-07-16

## Data We Collect

None. This extension does not collect, store, or transmit any personally
identifiable information, browsing history, web content, or user communications.

## Local Storage

The extension only uses `chrome.storage.local` to save its settings
(forward interval, interval mode, filter options, panel position, on/off state)
locally in your browser. This data never leaves your device and is never sent
to any server.

## Permissions

- **storage**: to persist the extension's settings locally.
- **access to `*.iqiyi.com`**: solely to read danmaku text and send danmaku on
  iQiyi video pages, which is required for the extension's core function. Access
  is strictly limited to the iQiyi domain.

## Third Parties

This extension contains no third-party analytics, advertising, or tracking code,
and loads no remote code.

## Contact

For questions, please open an issue on the project's GitHub repository:
https://github.com/wenzier/iqiyi-danmu-forwarder
