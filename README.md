# 爱奇艺实时弹幕转发器 (iQiyi Danmu Forwarder)

一个极简的 Chrome 扩展：在爱奇艺网页版视频中，每隔设定的秒数自动抓取屏幕上正在飘过的一条实时弹幕，原样再发送出去，让你的弹幕跟上大家的节奏。

只做转发这一件事，没有别的功能。

## 功能

- **定时转发**：每隔 N 秒（默认 20，最小 5）从屏幕上的实时弹幕里随机抽一条再发出。
- **过滤规则**：
  - 避免重复（最近 30 条内不重复转发）
  - 跳过超长弹幕（默认超过 20 字）
  - 跳过异常内容（含网址、纯表情、特殊字符占比过半）
- **兜底**：抓不到实时弹幕时，可选从内置短语里随机发一条，保证不断场。
- **浮动面板**：视频页右下角，可拖动、可折叠。运行状态持久化，重开自动恢复。
- **兜底选择器**：抓不到弹幕或发不出时，可分别粘贴“弹幕元素”和“输入框”的 CSS 选择器。

## 安装（加载已解压扩展）

1. 下载本文件夹到本地。
2. 打开 Chrome，访问 `chrome://extensions/`。
3. 打开右上角「开发者模式」。
4. 点击「加载已解压的扩展程序」，选择本文件夹。
5. 打开任意爱奇艺视频页（`*.iqiyi.com`），面板出现在右下角，点「开始转发」。

> 提示：转发前请先在播放器里打开弹幕栏，并确保弹幕处于开启状态（屏幕上要有别人的弹幕在飘，才能抓取）。

> ⚠️ 限制：只能读取以 DOM 元素渲染的弹幕。若播放器改用 `<canvas>` 绘制弹幕（文字是像素），DOM 层无法读取，将抓不到弹幕并走兜底。此时可尝试在面板里粘贴自定义弹幕元素选择器。
